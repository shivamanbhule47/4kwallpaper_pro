# 4KWALLPAPER Blog

A premium dark-mode editorial blog built with Next.js 15, powered by a GitHub-based CMS.

## GitHub CMS Workflow

1. Create a GitHub repository (e.g., `yourname/4kwallpaper-content`)
2. Create a `posts/` directory
3. Add Markdown/MDX files with frontmatter:

```md
---
title: "The Future of 4K Displays"
date: "2026-05-28"
category: "Technology"
excerpt: "Exploring next-generation display technology and its impact on digital art."
coverImage: "https://your-cdn.com/image.jpg"
featured: true
---

Your article content here...
